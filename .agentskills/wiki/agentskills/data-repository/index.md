# Directory Index: agentskills/data-repository

## Concepts

* [Architecture Decision Records (ADRs)](adr.md) - Decoupled, version-controlled records documenting key architectural choices, trade-offs, and consequences.
* [Episodic Memory Database](memory-db.md) - Local SQLite database storing unstructured facts, decisions, and preferences across agent sessions.
* [Glossary and Context (CONTEXT.md)](context.md) - Glossary defining terms and avoided vocabulary to ensure terminology alignment across sessions.
* [Namespaced Taskfiles](taskfiles.md) - Modular task runner configuration files declaring namespaced automation tasks for AI and UX.
* [OKF Wiki Root (wiki/)](wiki.md) - Central directory housing version-controlled, portable Knowledge Bundles conforming to OKF v0.1.
* [Product Requirements Documents (PRDs)](prd.md) - Standardized folders storing feature specifications and version-controlled workflow trackers.
* [Reference Sandbox](references.md) - Git-ignored directory used to store downloaded research papers, raw datasets, and external specs.
* [Structural Templates](templates.md) - Directory holding standardised skeletons and templates for PRD generation and UX component bootstrap.
* [Traceability Report](traceability-report.md) - Auto-generated markdown file tracing SysML baseline requirements to functional BDD scenario validations.

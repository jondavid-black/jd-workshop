# engineer-quality

The **`engineer-quality`** is a specialized engineering subagent.

## 1. Capabilities
- Audits security controls and searches for credentials or file leakages.
- Verifies overall test completeness and validates pipeline runs.
- Operates primarily during Phase 4 (Implementation).

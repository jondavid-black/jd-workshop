# engineer-ux

The **`engineer-ux`** is a specialized engineering subagent.

## 1. Capabilities
- Outlines user personas and maps visual journeys.
- Compiles modular React components and Storybook stories, tracing them directly back to SysML use cases.
- Operates primarily during Phase 2 (UX Definition).

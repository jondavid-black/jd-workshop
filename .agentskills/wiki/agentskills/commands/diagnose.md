# /diagnose Command

The `/diagnose` command triggers the structured, scientific debugging loop to investigate reported system bugs or performance regressions.

## Workflow

1. **Invokes `diagnose` Skill**: Systematically guides the agent through reproduction, minimization, hypothesis formation, instrumentation, and regression testing.
2. **Local Isolation**: Restricts investigations and experiments to isolated environments or test harnesses, preventing any unwanted workspace side effects.

# engineer-software

The **`engineer-software`** is a specialized engineering subagent.

## 1. Capabilities
- Implements backend APIs (Python/FastAPI) and frontends (React/TS).
- Develops code sequentially using strict Test-Driven Development (TDD) red-green-refactor loops.
- Operates primarily during Phase 4 (Implementation).

# Structural Templates

Templates are maintained under `.agentskills/templates/` to guide consistent, token-efficient creations:

## 1. Modular Skeletons
- **`templates/ux/`**: Skeletons for bootstrapping Storybook components, React stories, and UX designs.
- **`templates/prd/`**: Skeleton for drafting standard Product Requirements Documents (PRDs).

## 2. Capabilities
Enables agents and developers to clone clean, pre-configured boilerplate skeletons, saving token space and ensuring repository formatting parity.

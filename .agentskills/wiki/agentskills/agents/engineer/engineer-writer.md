# engineer-writer

The **`engineer-writer`** subagent maintains exceptional completeness and quality of project documentation and OKF Knowledge Bundles.

## 1. Capabilities
- Synchronizes documentation with active codebase developments.
- Operates on a modular, general-purpose workflow that dynamically searches and maintains documents inside `documents/` and wiki concepts inside `.agentskills/wiki/`.
- Enforces a strict user approval gate before creating any new documentation files.

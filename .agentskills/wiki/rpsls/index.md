# Directory Index: rpsls

## Concepts

* [Backend FastAPI Server](backend_api.md) - Stateless FastAPI server endpoints with CORS enabling and play routing.
* [Backend Game Engine](backend_engine.md) - Core game logic module containing choice options, winning rule definition, and choice outcome evaluation function.
* [Frontend React Game App](frontend_app.md) - Strictly typed React application featuring countdown mechanics, round selection, dueling reveal states, and Match scoring.
* [Go-Task Orchestration Configuration](orchestration.md) - Local orchestration definitions using root-level Taskfile.yml with build, up, down, and validation tasks.
* [Kubernetes Pod Configuration](pod_manifest.md) - Local multi-container Kubernetes Pod configuration exposing separate host ports for the frontend and backend.

# Glossary and Context (CONTEXT.md)

The **`CONTEXT.md`** file is the system-wide single source of truth for terminology and domain definitions within AgentSkills.

## 1. Capabilities
- **Terminology Alignment**: Formally defines terms such as `Agent Configuration`, `Skill`, `Knowledge Bundle`, `Remember`, `Recall`, and `Conflict Resolution`.
- **Glossary Checks**: Checked by analysts and planners during Phase 1 to prevent term drift or overloaded phrases.
- **Avoidance Vocabulary**: Explicitly specifies terms to avoid to guarantee consistent agent understanding.

# Directory Index: agentskills/commands

## Concepts

* [/diagnose Command](diagnose.md) - Systematically reproduces, diagnoses, and instruments bugs and performance regressions.
* [/pr-done Command](pr-done.md) - Resets and cleans up the local target workspace after a Pull Request is successfully merged.
* [/refine Command](refine.md) - Starts a new manual requirement refinement session to stress-test plans and clarify domains.
* [/refine-issue Command](refine-issue.md) - Starts a new requirement refinement session driven by a specific GitHub issue description.
* [/reflect Command](reflect.md) - Analyzes conversation history to identify opportunities for improving agent definitions and skills.
* [/resolve-pr-comments Command](resolve-pr-comments.md) - Pulls unresolved review comments from GitHub PRs, implements fixes, and post replies.
* [/sync Command](sync.md) - Synchronizes agent configurations, skills, and task files from AgentSkills into a Target Repository.
* [/write-skill Command](write-skill.md) - Launches the 'write-a-skill' helper to securely author or refine custom agent skills.

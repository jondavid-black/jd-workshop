# continuous-learning Skill

The **`continuous-learning`** skill is designed to help agents analyze interaction histories, recognize user feedback/corrections, and propose system prompt updates to avoid repeating errors.

## Key Actions

- **History Scanning**: Retrieve and process full-turn interaction histories.
- **Feedback Identification**: Parse user critiques or corrections to identify patterns.
- **Prompt Refinement**: Generate and output improved system prompt drafts in accordance with template rules.

# Directory Index: agentskills/tooling

## Concepts

* [Bun TS/JS Runtime](bun.md) - High-performance TypeScript and JavaScript compiler, bundler, and test runner used for helper scripts and unit tests.
* [Go-Task Task Runner](go-task.md) - Command-line task runner executed via 'task', defining modular, platform-agnostic workspace automation pipelines.
* [Memory CLI Tool (memory_cli.ts)](memory-cli.md) - Custom TypeScript CLI tool providing direct SQLite query execution for episodic memory add, search, and delete actions.
* [OKF CLI Tool (okf_cli.ts)](okf-cli.md) - Custom TypeScript CLI tool built to manage, lint, index, search, and parse changelogs for OKF Knowledge Bundles.
* [Prompt Validation Tool (validate_prompts.py)](validate-prompts.md) - Custom Python script used to lint and validate agent profiles and skill markdown files for compliance.
* [Systems Traceability Auditor (validate_traceability.py)](validate-traceability.md) - Custom Python script used to audit and map SysML requirements coverage against BDD tests.
* [UV Python Environment Manager](uv.md) - Ultra-fast Python package and environment manager used to run formatting, lints, and behavior tests.

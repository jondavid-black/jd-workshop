# /refine Command

The `/refine` command launches an interactive, systematic refinement interview between the agent and the user.

## Purpose

- **Ambiguity Resolution**: Eliminates fuzzy or overloaded terminology.
- **Glossary Synthesis**: Dynamically compiles and aligns `.agentskills/CONTEXT.md`.
- **Vertical-Slice Planning**: Outlines structured implementation tasks with explicit user sign-off.
